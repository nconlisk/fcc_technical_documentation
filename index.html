Introduction

Python is a powerful object-oriented programming language, comparable to other high level scripting languages such as Perl, Ruby, or even Java. Some of Python's notable features:

  • Uses a syntax that is easy to read and write.
  • Comes with a large standard library covering most development needs.
  • Rapid testing and prototyping can be done with Python's interactive mode.
  • Can be easily extended by adding new modules implemented in languages such as C/C++.
  • Cross-platform support for all major operating systems.

Python code is governed by style rules (PEP). You can test, check, and improve your code online using resources such as Pythonchecker.com.


Installation

Before you start, you will need Python on your computer. You can use the command line window to check if an up to date version of Python is already installed on your system. If you see a response from a Python interpreter it will include a version number in its initial display. Generally any Python 3.x version will do. If python starts a Python 2.x interpreter, try entering python3 and see if an up to date version is already installed, otherwise it may be necessary to download the latest version from the Python repository.

If no install is detected on your system, you need to install Python by naivgating to the download section of python.org. Installers exist for all major operating systems. Select the one most appropriate for your system.


First programs

Writing your first program may seem daunting, however, Python makes this a breeze. Fire up your Python interpreter and type in the below code: >>>print("Hello World")
Hello World Then press return to execute the command.

Running Python code from a script file: While simple commands can be run from a terminal or within the interpreter, more complex programs may include several commands and variables. To conduct more advanced operations we can save a series of commands into a Python script file.
We will now recreate the "Hello World" code in script form. On the menu of the python interpreter click File > New File. Now copy and paste the previous command into the new file window that has opened up: message = "Hello World"

print(message)
Save this file as hello_world.py. We can then press f5 or click the run button from the menu bar to execute the commands in this script. As before the interpreter will output: >>>
Hello World
For more details of how to call the script file from the command line, please consult the Python documentation or one of the referenced guides below.

Another fun beginner feature of the Python language is the turtle library. This library is great for kids and beginners alike and lets you really visualise the code you are writing via a small turtle on the onscreen canvas. To get started with your turtle, type the following into the interperter: import turtle
s = turtle.getscreen()
t = turtle.Turtle()
t.shape("turtle")
 Now that you have a turtle on your screen, we need to move it somehow. Luckily, their are some handy build in functions we can use to get started: t.right(45)
t.forward(150)
t.left(60)
t.backward(300)
 Try playing around with these and other commands to get a sense of what edits to your code can do. To learn more about the features and what can be done with the turtles library, please consult the turtles documentation.
